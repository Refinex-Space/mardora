# Security Policy

If you discover a security vulnerability in this project, please report it responsibly.

We take all security reports seriously and will investigate promptly to address any valid concerns.

## Reporting a Vulnerability

Please use GitHub's private vulnerability reporting feature to securely disclose potential security issues.

**[Report a vulnerability](https://github.com/neuronexul/draftly/security/advisories/new)**

This ensures sensitive information stays private until a fix is available.
